# clambback Brand Assets

This directory contains the brand icons for `clambback`.

## Files

- `clambback-official-logo.png` - official generated logo for GitHub and project presentation.
- `clambback-squircle.svg` - glossy macOS-style squircle app icon (backpack motif, no wordmark).
- `clambback-squircle-1024.png` - 1024x1024 PNG render of the squircle icon.

## Usage

- Use `clambback-official-logo.png` as-is for the repository README and project profile surfaces. Do not crop out the backpack, alter the wordmark, add shell/clam motifs, or add C++ shield decorations.
- Use `clambback-squircle.svg` (or the PNG render) for app grids, product cards, favicon-style placements, and anywhere a square icon tile is needed.
